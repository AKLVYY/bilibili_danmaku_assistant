# 修改日志

## 2026-03-30

### 已修复

- 修正源码启动导入路径，统一 `ui_codes` 模块引用，并补充打包配置。
- 为登录、弹幕、点赞接口增加超时、HTTP 状态和 B 站业务返回码校验。
- 修复编辑任务时先写入后校验导致的脏数据问题。
- 修复取消定时后时间控件未恢复可编辑的问题。
- 为二维码轮询增加并发保护，避免慢网下重复堆积轮询线程。
- 为空任务队列增加发送前拦截，避免无意义执行。

### 本次增强

- 新增任务配置结构校验：
  - 配置文件必须是 JSON 数组。
  - 每条任务必须包含合法的 `room`、`msg`、`loop`、`send_interval`。
  - `loop` 不能小于 `1`，`send_interval` 不能小于 `5`。
  - 缺少或空白的 `uname` 会自动补成 `房间 {room}`。
- 新增 B 站错误码中文化：
  - 常见登录态、权限、风控、二维码状态错误会转换成更易读的中文描述。
  - 未知错误码也会保留原始信息，方便排查。
- 新增更友好的网络错误日志：
  - 请求超时、连接失败、HTTP 错误会转成中文提示。
